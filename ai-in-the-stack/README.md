# AI in the Stack - Code Labs

Hands-on code examples from the [AI in the Stack](https://pipelineandprompts.com/series/ai-in-the-stack/) series.

## Labs

1. [AI Tooling Evaluation Framework](01-ai-tooling-evaluation/) - Evaluate LLM APIs for platform engineering
2. [RAG Runbook Assistant](02-rag-runbook-assistant/) - Build a semantic runbook search with FastAPI + Chroma
3. [MCP Server for Kubernetes](03-mcp-for-kubernetes/) - Deploy an MCP server to query K8s resources
4. [Prompt Versioning CI for OpenShift](04-prompt-versioning-ci/) - Schema validation and governance for LLM prompts in CI/CD

## Prerequisites
- Python 3.11+
- Docker
- Kubernetes cluster (for lab 3)
